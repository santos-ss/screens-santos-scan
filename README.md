# 🔍 ScreeS | Santos Scanner

Scanner forense Android para análise de:

- Root
- Kernel
- Processos suspeitos
- Arquivos suspeitos
- Conexões ADB

## 🚀 Instalação (Termux)

```bash
pkg update && pkg upgrade -y
pkg install git -y

git clone https://github.com/santos-ss/screens-santos-scan.git
cd screes-santos-scan

chmod +x scan.sh
